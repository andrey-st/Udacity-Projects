Important HTML and CSS Conepts

Lesson 1: The Basics of the Web and HTML

What is the Web?

The Web is composed largely of HTML documents, which are used by browsers to interpret web pages.

Web consists of the following elements:

  • Comptuer & Browser
  • Internet
  • HTTP
  • Servers

HTML Elements

HTML elemets are used to format the way content is displayed on a web page, or to include external content in the page. Most elements consist of opening and closing tags, whereas others don't have closing tag (since they don't have content, and are called void elements. Examples:

  • <p>content</p>
  • <img src ="...">

HTML Attributes

HTML Attributes are assigned to elements and add further information. Examples:

  • <a href="www.google.com"> text </a>
  • <img src="www.test.com/img" alt = "no such pic">

Inline and Box Container Elements

Inline container elements does not create distinct "boxes" on the page. Such elements are:

  • <br>
  • <span>content</span>

Box Container Elements define "designated areas". Such elements are:

  • <p>content</p>
  • &l;tdiv>content</div>

Lesson2: Creating a Structured Document

Two Main Types of Elements

HTML Elements are either inline, or a block. Blocks form an "invisible" rectangle around the content included in them.

Tree-like structures

Thanks to the block elements html documents can form a tree-like structure - in each block can be included multiple sub-blocks, in which in turn can be included sub-sub-blocks, etc. Creating these blocks and sub-blocks is refered as Boxifying.

HTML, CSS, DOM

HTML is the language of the Web - it sets the structure of a web page. The browser uses the Document Object Model (DOM) to visualize the tree-like structure of the web page. CSS is used to define the style of the content within HTML - font, color, borders, etc.

Relationship between HTML and Boxes

The text version of the HTML documents are structured with changing identations in order to reflect the tree-like structure of the HTML document.

Text Editors that Help Write HTML Code More Efficiently

Text editors make life easier - for example some text editors automatically generate a closing HTML tag when you write an opening tag.

Lesson 3: Adding CSS Style and HTML Strcuture

What is CSS?

CSS code is used to add style to HTML documents. CSS code can be added in the <head> tag of an HTML document, as attributes in tags, or in a separate .css document. If CSS is added in a separate document, it has to be called from the <head> tag in the html document using the <link> tag.

How does CSS works?

CSS assigns style to the html document using the so-called selectors. Selectors specify to which element in the html document the style should be applied. Thus, the following terms are important to be remembered:

  • Rule: a line of CSS code describing the value that a certain attribute should take.
  • Property: The property you want to change.
  • Value: The value that you want to assign to the attribute.
  • Selector: The name that you use to in order to target the elements that are assigned to a class or id attribute in the HTML.
  • Class: A class refers to a group of elements that can be styled together. Class names should not contain periods or any other punctuation marks such as class="1.1"
  • ID: ID's are unique identifiers that uniquely identifies an element in HTML.

The Box Model

HTML elements are boxes, which consist of four elements:

  • Content - pure content size, e.g. text/image dimensions
  • Padding - space between the border and the content
  • Border - border of the Box
  • Margin - space between the border and other Boxes

Width of a box equals the sum of content, padding and border widths. This makes manupilation of elements tough. In order to make it easier, there are two techniques used to help deal with sizing issues:

  1. Set sizes in terms of percentages rather than pixels; add max-width
  2. Set the box-sizing attribute to border-box for every element - this keeps the width of the box the same regardless of the changes made to the border and the padding. (browser-specific prefixes should be added with this)

Box Positioning

Divs are block elements (as opposed to inline), so by default they take up the entire width of a page. Adding the rule display: flex; to the appropriate CSS will override this behavior and let divs appear next to each other. To use this size of the box should be smaller than the default 100%.
